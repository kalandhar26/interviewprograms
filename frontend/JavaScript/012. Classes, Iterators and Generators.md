## Classes

- A class is a blueprint used to create objects with shared properties and methods.
- From one blueprint, we can build many houses. That blueprint is a Class .The houses are Objects.

```javascript
class Animal {
  constructor(name) {
    this.name = name;
  }

  speak() {
    console.log(this.name + " makes a sound");
  }
}

const dog = new Animal("Dog");
dog.speak(); // Dog makes a sound
```

- Classes are syntactic sugar over prototypes
- Methods go to prototype automatically
- Cleaner and readable than old prototype syntax

## Iterators

- An iterator is an object that provides a way to access elements one by one using a next() method.
- You have a box of chocolates. You eat one chocolate at a time, not all at once.
- That step-by-step eating is iteration. The rule that gives next chocolate is Iterator.

```js
const arr = [10, 20, 30];

const iterator = arr[Symbol.iterator]();

iterator.next(); // { value: 10, done: false }
iterator.next(); // { value: 20, done: false }
iterator.next(); // { value: 30, done: false }
iterator.next(); // { value: undefined, done: true }
```

### Iterator Rules

- An iterator must return

```
{ value: something, done: true/false }
```

- for...of uses iterators internally

```javascript
for (let num of arr) {
  console.log(num);
}
```

## Generators

- A generator is a special function that can pause and resume execution using the yield keyword.
- Imagine a magic book,You open it…It gives one story, Then pauses, You say “next!”, It continues from where it stopped.
  That magic book is a Generator.

```javascript
function* numbers() {
  yield 1;
  yield 2;
  yield 3;
}

const gen = numbers();

gen.next(); // { value: 1, done: false }
gen.next(); // { value: 2, done: false }
gen.next(); // { value: 3, done: false }
gen.next(); // { value: undefined, done: true }
```

### Generator Facts

✔ Uses function*
✔ Uses yield instead of return
✔ Automatically creates an iterator

## Async Iterators & Async Generators

- An async iterator allows iterating over data that arrives asynchronously, returning a Promise for each value.

```javascript
const asyncData = {
  data: [1, 2, 3],
  async *[Symbol.asyncIterator]() {
    for (let item of this.data) {
      await new Promise(res => setTimeout(res, 1000));
      yield item;
    }
  }
};

(async () => {
  for await (let value of asyncData) {
    console.log(value);
  }
})();
```

- An async generator is a function that produces values asynchronously and can pause using await and yield.

```javascript
async function* fetchNumbers() {
  yield 1;
  await new Promise(res => setTimeout(res, 1000));
  yield 2;
  await new Promise(res => setTimeout(res, 1000));
  yield 3;
}

(async () => {
  for await (let num of fetchNumbers()) {
    console.log(num);
  }
})();
```
