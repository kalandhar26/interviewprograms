# Module

- A module is a reusable piece of code that encapsulates variables, functions, or classes.
- Modules help to organize code into reusable and independent units.

## Types of Module Systems

- JavaScript has two main module systems:
- CJS (CommonJS) – Node.js (old style)
- ESM (ES Modules) – Modern JavaScript (standard)

## 1. CJS

- CommonJS is a module system where modules are loaded using require() and exported using module.exports.

## 2. ESM

- ES Modules is the official JavaScript module system using import and export.
- Export Types in ESM
- Named Export, Default Export

# Module:

- A module is a reusable, encapsulated unit of code.

## CommonJS:

- CommonJS uses require and module.exports, mainly in Node.js.

## ES Modules:

- ES Modules use import/export and support static analysis and tree shaking.

## Named Export:

- Exports multiple values by name.

## Default Export:

- Exports a single main value.
