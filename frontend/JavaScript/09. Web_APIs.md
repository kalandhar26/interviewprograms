# Web APIs

- Web APIs are browser-provided interfaces that allow JavaScript to interact with the browser and the outside world.
- Web APIs are NOT part of JavaScript. Provided by Browser

## 1. DOM

- The DOM is a programming interface that represents HTML as a tree of objects.
- DOM Can Change text, Change style, Add/remove elements, Handle events.

## 2. XHR and Fetch

- XHR is an older API used to make HTTP requests from the browser.
- Fetch is a modern API for making HTTP requests that returns Promises.

## 4. Storage

- Local Storage, Session Storage, Cookies
- Local Storage is used to store data on the client side in the browser. (localStorage)
- Session Storage is used to store data for the duration of the page session. (sessionStorage). Cleared when tab closes.
- Cookies are used to store small amounts of data on the client side in the browser. (document.cookie) Sent with every
  request and small size.

## 5. Video and Audio

- JS gets remote control. To play, pause, mute media.

```html

<video id="vid" src="movie.mp4"></video>
```

```javascript
const video = document.getElementById("vid");
video.play();
video.pause();
video.volume = 0.5;

```

- Media API Can
  ✔ Play / Pause
  ✔ Control volume
  ✔ Seek time
  ✔ Fullscreen

## 6. Drawing Graphics.

- Canvas API allows JavaScript to draw graphics using pixels.
- WebGL API allows JavaScript to render 3D graphics in the browser.
