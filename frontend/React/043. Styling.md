# CSS Modules

- Imagine each child has their own locker. No one can touch anotherâ€™s stuff.
- CSS Modules avoid class name conflicts.
- CSS Modules scope CSS locally by default, preventing global namespace collisions.

```css

  color: red;
}
```

```jsx

<button className={styles.button} />
```
