## Refs

- Refs provide direct access to DOM nodes or component instances, bypassing React's render cycle. Use useRef hook in
  functions.
- Ref is a concept in React that lets you access or store something without causing a re-render.
- Access DOM elements
- Store mutable values that donâ€™t affect UI
- Interact with child components or external libraries

```jsx
import { useRef } from 'react';

function TextInput() {
  const inputRef = useRef(null);

  const focusInput = () => inputRef.current.focus();

  return (
    <>
      <input ref={inputRef} />
      <button onClick={focusInput}>Focus</button>
    </>
  );
}
```
